What are Databases?

A structured set of data

Databases are a collection of information. They are used for creating, manipulating and deleting data. There are many different types of databases. They are classified in a basic approach: relational database or non-relational database. Relationship means a set of our data is related to another section of our data. Meanwhile, in a non-relational database, there is not a common related section of our data. Our databases are put into sections of data called schemas. Within each schema, data sets are organized into tables. Those tables contain columns and rows to house the particular set of data.

Interaction

The amazing part about having a Database is that it is a storage bin that holds onto all saved data. well if only one user had access to that data - much like what we have built into our computers, it wouldn't be useful to other users. However, the joy of having a database is that is is a storage bin manay users can interact with: creating, reading, updating and deleting information that all users would see.
