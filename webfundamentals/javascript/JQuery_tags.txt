Manipulation (retrieving or setting value or text in any HTML element)

.after() || Description: Insert content, specified by the parameter, after each element in the set of matched elements.

.prepend() || Description: Insert content, specified by the parameter, at the beginning (first child) of each element in the set of matched elements.

.attr() || Get's the value of attribute for the first element in a set of matched elements.(.attr()). To set an attribute, pass both the property name and value eg: (.attr(attributneName, Value)).

.before() || Insert contents via a parameter before an element.

.html() || Getter/Setter: Get html elements without passing anything through the parameter. Or set HTML contents via parameter. HTML tags can be passed directly through the parameter.

.text() || Return or set texts of an element.

.val() || Return or sets value.
